

## command

install 
```
pnpm i 
```

run dev
```
pnpm run dev
```



## タイムライン

1. デザインの確認
   1. 使用しているフォントの確認
      1. font-family
      2. font-weight
   2. デザインカンプの横幅・コンテンツ幅
   3. 画像の書き出し
2. HTML, CSSの実装
3. パフォーマンスの確認(lightHouse)
   1. 画像の圧縮
